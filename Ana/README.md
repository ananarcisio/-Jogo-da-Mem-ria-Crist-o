# ✝️ Jogo da Memória Cristão

Um jogo de memória inspirador com símbolos cristãos e versículos bíblicos, desenvolvido com HTML, CSS e JavaScript.

**Projeto criado usando Amazon Q Developer CLI no Hacktown 2025**

## Funcionalidades

- 🎮 Jogo de memória 4x4 com 8 pares de símbolos cristãos
- ✝️ Símbolos: Cruz, Oração, Coração, Pomba, Estrelas, Bíblia, Música
- 📖 Versículos bíblicos inspiradores aleatórios
- ⏱️ Contador de tempo e movimentos
- 🎵 Efeito sonoro de vitória
- 📱 Design responsivo para mobile e desktop
- 🎨 Interface moderna com glassmorphism

## Como jogar

1. Abra o arquivo `index.html` no navegador
2. Clique nas cartas para virá-las e encontrar os pares
3. Memorize a posição dos símbolos
4. Complete todos os 8 pares para vencer
5. Leia os versículos bíblicos para inspiração
6. Use "Novo Jogo" para recomeçar

## Símbolos do Jogo

- **✝️** - Cruz (símbolo da fé cristã)
- **🙏** - Oração (comunicação com Deus)
- **❤️** - Amor (amor de Deus)
- **🕊️** - Pomba (Espírito Santo)
- **⭐** - Estrela (luz de Cristo)
- **🌟** - Estrela Brilhante (esperança)
- **📖** - Bíblia (Palavra de Deus)
- **🎵** - Música (louvor e adoração)

- Animações suaves de flip das cartas
